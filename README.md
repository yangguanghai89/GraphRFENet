# GraphRFENet

需新建save和za两个文件

使用train.py开始训练模型，模型文件为model.py，选取最优的模型使用break.py进行测试

数据集链接（夸克网盘）：https://pan.quark.cn/s/e655fc579dac 数据集下载后在tool/utils文件中更改文件路径

图数据链接（夸克网盘）：https://pan.quark.cn/s/374708dbf9d2 数据下载后与train.py文件在同一级目录

如果想自己构图，使用tool\build_graph.py文件构图

预训练语言模型链接（bert和bge）：https://pan.quark.cn/s/3da9adfbb82d
